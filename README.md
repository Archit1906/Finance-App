# Wealth OS: Financial Engineering System

Welcome to the **Wealth OS**, a futuristic, high-performance financial management terminal designed to bring enterprise-grade aesthetics and functionality to personal finance tracking. 

Built with an intense cyber-thematic aesthetic, Wealth OS acts as your personal command center for tracking Net Worth, Goals, Budgets, and structural financial data.

---

## 🎨 Key Interface Architecture
### Holographic Command Center Redesign
The entire platform is engineered around a "Deep Space / Terminal" glassmorphism aesthetic. It utilizes dynamic `.bg-vault-blueprint` grids, high-contrast cyber-cyan highlights, deep reds for critical actions, and terminal-style monospaced data rendering (`JetBrains Mono` / `Fira Code`). 

### Entry Portal 2.0 (Dual-Mode Access)
The login experience has been transformed into a fully interactive Entry Node:
- **MODE_001 [SECURE LOGIN]**: Standard secure access to existing personal financial nodes.
- **MODE_002 [NODE PROVISIONING]**: Dynamic user registration protocol simulating account creation with terminal metrics, fake 2FA toggles, and currency selection logic. 
- Integrated with complex 3D page-flip animations and gold-dust transition pipelines.

### Dynamic Dashboard HUDs
Real-time SVG-based ring components tracking your overall "Health Score," automated number counters, and deep-space atmospheric breathing gradients applied globally.

## 🚀 Getting Started

Wealth OS is a Full-Stack application split into separate `frontend` and `backend` repositories inside this workspace.

### Prerequisites
- [Node.js](https://nodejs.org/en) (v18+ recommended)
- `npm`

### 1. Launching the Backend API
The core API manages user registration logic and provides dynamic reporting to the terminal.
```bash
cd backend
npm install
npm run dev
```

### 2. Launching the Frontend Terminal
The visual terminal is built using Vite + React. 
```bash
cd frontend
npm install
npm run dev
```
*(The terminal will default to running on `localhost:5173` or similar via Vite.)*

---

## 🛠️ Stack & Technologies
* **Frontend**: React, Vite, TailwindCSS (for complex UI transitions, layout, and deep-theming), Lucide-React (for terminal iconography).
* **Backend**: Express / Node.js, routing securely to TanStack Query calls on the front. 
* **Design Pattern**: Glassmorphism, Brutalist Terminal styling.
